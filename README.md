## Running the program
1. Download the github repo
2. Unzip the folder
3. Open terminal and cd into the unzipped folder (i.e. `cd Desktop/Valentine-Heart-main`
4. Compile the program `javac Heart.java`
5. Run the program `java Heart [Name]`
